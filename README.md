# Православни календар 2026 (Orthodox Calendar 2026)

A beautiful, modern, and fully responsive Orthodox Church Calendar for 2026, featuring the Serbian Orthodox liturgical calendar with fasting guidelines, feast days, and daily saints' commemorations.

![License](https://img.shields.io/badge/license-MIT-purple)
![PWA](https://img.shields.io/badge/PWA-enabled-blue)

## ✨ Features

- 📅 **Complete 2026 Calendar** - All 12 months with Orthodox feast days and fasting information
- 🎨 **Beautiful Purple Design** - Elegant gradient design with smooth animations
- 🌓 **Light & Dark Mode** - Toggle between themes with persistence
- 📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices
- 🖨️ **Print-Friendly** - Optimized print layout for physical calendar
- 📲 **PWA Installable** - Install as a Progressive Web App on any device
- 🇷🇸 **Serbian Cyrillic** - Full support for Serbian language and Cyrillic script
- 🙏 **Fasting Guide** - Visual indicators for fasting levels (strict, oil & wine, fish allowed)
- ⚡ **Fast & Lightweight** - No external dependencies, works offline

## 📖 Calendar Features

- **Feast Days** - Highlighted major Orthodox feasts
- **Fasting Days** - Color-coded fasting levels
- **Saints' Days** - Daily commemorations of saints
- **Old & New Style** - Both Julian and Gregorian dates
- **Special Notes** - Church holidays and observances

## 🚀 Usage

1. **Online**: Simply open `index.html` in any modern web browser
2. **Install as App**: Click the install button in your browser to add to home screen
3. **Print**: Click the "Штампај" (Print) button for a printer-friendly version
4. **Theme**: Toggle between light and dark modes using the theme button

## 📱 Installation as PWA

1. Open the calendar in Chrome, Edge, or Safari
2. Click the install icon in the address bar (or "Add to Home Screen" on mobile)
3. Access the app offline from your home screen

## 🎨 Color Legend

- **⬤** - Strict fasting (no food, oil, or wine)
- **◐** - Oil and wine allowed
- **◑** - Wine and oil allowed
- **ⵔ** - Fish allowed

## 🛠️ Technical Details

- **Pure HTML/CSS/JavaScript** - No frameworks or build tools required
- **Service Worker** - Offline functionality and caching
- **CSS Grid & Flexbox** - Modern, responsive layout
- **CSS Variables** - Easy theme switching
- **Base64 Manifest** - Embedded PWA configuration
